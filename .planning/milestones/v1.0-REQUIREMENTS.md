## Requirements

### Functional Requirements

- [x] Moteur d'analyse forensique (`NilinkVerifierEngine`) capable de traiter images et flux video
- [x] Detecteur ELA (Error Level Analysis) pour inpainting et zones generees
- [x] Detecteur FFT (analyse spectrale) pour artefacts GAN/diffusion
- [x] Detecteur rPPG (photoplethysmographie) pour liveness/deepfake sur visages
- [x] Detecteur d'upscaling/restauration IA
- [x] Detection de visage integree (OpenCV Haar/DNN)
- [x] Output structure : score de confiance (0-1), liste d'anomalies, heatmap (image + coordonnees)
- [x] Performance temps reel : 15+ FPS sur flux video
- [x] Gestion du frame dropping pour maintenir la latence
- [x] API REST pour analyse d'images individuelles
- [x] API WebSocket pour flux video continu
- [x] Mode demo webcam pour validation
- [x] Suite de tests avec images connues (vraies + fakes)

### Out of Scope (v1.0)

- Modeles Deep Learning lourds (PyTorch/TensorFlow)
- Interface utilisateur graphique
- Analyse audio/voix
- Stockage/historique des analyses
- Authentification/facturation API
